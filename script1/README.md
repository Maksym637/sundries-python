### Description

Implementation of a strategy design pattern that performs text comparison from two sources (HTML and PDF) using the `cosine similarity` metric

---

### Execution

1. Install all dependencies using the command below:

```bash
poetry install
```

2. Activate poetry using the command below:

```bash
poetry env list --full-path
```

```bash
source [poetry env path]/bin/activate
```

3. Go to the `script1` dir and execute the command below:

```bash
python main.py
```
